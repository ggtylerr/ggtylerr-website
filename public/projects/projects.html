mutat.io

The AIO Conversion tool.

Able to convert units, files, timezones, and even languages, it's the most robust conversion tool available. Complete with an open source API and documentation. Currently in very early alpha and only unit conversion is available at this time.

DSA Bot

The Discord bot made for esports.

Starting work in early 2020, it grew to a reliable bot for both esports and general purpose needs. It was originally made for the DSA Esports club, and is mostly public domain. The only thing that isn't is the hosting server. Everything else is fair game.

Website (Contains other links)

Dark Side SCP

An actual working dark mode for the SCP Wiki

Tired of having to use oddball extensions and shady styles just to not have eyes burnt? Well, here's a viewer specifically made for the SCP wiki, with an actual working dark theme. You're welcome.

(P.S. SCP Wiki please add a dark theme thanks :>)
